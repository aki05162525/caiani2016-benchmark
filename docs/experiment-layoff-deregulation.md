# 解雇規制緩和シミュレーション実験設計

## 1. 研究の背景と問題意識

### 1.1 社会的背景
日本では、正規雇用の解雇規制が厳しく、労働市場の硬直性が指摘されている。一方で、解雇規制緩和に対しては、雇用不安の増大や消費抑制といった懸念も存在する。

### 1.2 研究の位置づけ
Caiani et al. (2016) のマクロ経済ABM（エージェントベースモデル）を拡張した二重労働市場モデルを用いて、**解雇規制緩和がマクロ経済変数に与える影響**をシミュレーション分析する。

### 1.3 先行研究との関係
- Caiani et al. (2016): Stock-Flow Consistentなマクロ経済ABMのベースライン
- 本研究: R（正規）/N（非正規）の二重労働市場構造を導入し、解雇規制のパラメータ化を実現

---

## 2. 検証仮説

### 主仮説
> **解雇規制の緩和（正規労働者の解雇率引き上げ）は、短期的に労働市場の柔軟性を高めるが、勤続年数の短縮を通じた人的資本蓄積の阻害により、長期的なマクロ経済パフォーマンスを低下させる。**

### 具体的な予測
| 変数 | 予測される変化 | 経路 |
|------|---------------|------|
| 平均勤続年数 | **減少** | 解雇増 → 雇用関係の短期化 |
| 生産性 | **低下** | 勤続年数↓ → tenure productivity効果↓ |
| GDP | **低下** | 生産性↓ + 消費↓ |
| 消費 | **減少** | 失業増 → 所得減 → 消費減 |
| 倒産件数 | **増加 or 減少?** | 労務コスト柔軟化 vs 需要減少の競合 |

---

## 3. シミュレーション設計

### 3.1 モデルの主要特徴
- **二重労働市場**: R（正規）とN（非正規）の2タイプの労働者
- **解雇率パラメータ**: `layoffRateR`, `layoffRateN` で解雇の容易さを制御
- **勤続年数と生産性**: tenure productivity機能で勤続年数→生産性向上を実装
- **CES生産関数**: R/N労働の代替弾力性を考慮

### 3.2 実験条件

| パラメータ | ベースライン | シナリオA |
|-----------|-------------|----------|
| `layoffRateR` | 0.3 (30%) | **0.5 (50%)** |
| `layoffRateN` | 0.8 (80%) | 0.8 (80%) |
| その他 | 同一 | 同一 |

- **比較方法**: 静的比較（各条件で別々にシミュレーションを実行し、定常状態を比較）
- **シミュレーション期間**: T=500期（定常状態に達するまで）
- **seed数**: 複数seedで実行し、統計的な頑健性を確認

### 3.3 観察変数

| 変数 | 計測方法 | 対応する仮説 |
|------|---------|-------------|
| 平均勤続年数（R労働者） | R労働者のtenure平均 | 主仮説の直接的検証 |
| GDP | 名目GDPの推移 | マクロ経済パフォーマンス |
| 消費 | 家計消費の推移 | 需要サイドへの影響 |
| 倒産件数 | 期間中の企業倒産数 | 企業の財務健全性 |
| 失業率（R/N別） | 各タイプの失業率 | 労働市場への直接的影響 |

## 4. モデルにおける因果経路

```
解雇規制緩和 (layoffRateR↑)
    │
    ├─→ 経路A: 人的資本蓄積阻害
    │       │
    │       ├─→ 解雇増加
    │       │       ↓
    │       ├─→ 平均勤続年数↓
    │       │       ↓
    │       └─→ tenure productivity↓ → 生産性↓ → GDP↓
    │
    ├─→ 経路B: 需要減少
    │       │
    │       ├─→ 失業増加 → 所得減少 → 消費↓
    │       │       ↓
    │       └─→ 需要減少 → 企業収益↓ → 倒産増?
    │
    └─→ 経路C: 労務コスト柔軟化（相殺効果）
            │
            └─→ 不況時の調整コスト↓ → 企業存続力↑ → 倒産減?
```

## 5. 本モデルの限界と留意点

### 5.1 未実装のチャネル
- **将来不安→予防的貯蓄**: 雇用不安の増大が直接的に消費を抑制するチャネルは未実装
  - 現状: 失業→所得減→消費減という**間接的経路のみ**
  - 影響: 消費抑制効果が過小評価される可能性

### 5.2 パラメータ設定の妥当性
- layoffRate = 0.3 → 0.5 の変更が「解雇規制緩和」としてどの程度現実的かは議論の余地あり
- 日本の解雇規制の実態との対応関係は定性的

### 5.3 外的妥当性
- ABMの結果は特定のパラメータ設定に依存
- 複数seedでの頑健性確認が必須
